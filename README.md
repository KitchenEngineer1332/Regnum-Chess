# Regnum Chess: A Premium Chess Experience
Regnum Chess is a web-based chess application that provides a premium gaming experience with its advanced AI evaluation, interactive UI, and seamless animations. The project aims to create a realistic and engaging chess environment for players of all levels, from casual enthusiasts to professional grandmasters. With its robust game engine and intuitive interface, Regnum Chess is the perfect platform for anyone looking to improve their chess skills or simply enjoy a game with friends.

## Key Features
* Advanced AI evaluation using positional tables for accurate piece placement and move prediction
* Interactive UI with animations and visual effects to enhance the gaming experience
* Support for single-player and multiplayer modes, allowing users to play against the AI or other players
* Customizable theme and layout options to suit individual preferences

## Tech Stack
* Frontend: HTML, CSS, JavaScript
* Backend: None (client-side application)

## Usage
1. Click on https://kitchenengineer1332.github.io/Regnum-Chess/
2. Click on the "Single Player" or "Multiplayer" button to start a new game
3. Use the mouse and keyboard to interact with the game board and pieces
4. Customize the theme and layout options by clicking on the gear icon
